# 白城主 (BaiChengZhu)
### 本地优先 · 可控云端API · 对话框式AI电脑控制助手
**轻量隐私化的多模态AI工具，适配8G内存/10G硬盘普通设备，无广告、无捆绑、无冗余后台进程**
> 核心定位：用对话框指令直接控制电脑 + 轻量化视频镜头文字分析，本地纯离线运行，云端API按需开启，兼顾隐私、性能与实用性

---

## 🚀 核心亮点（GitHub同类项目独一份）
1.  **「本地优先+可控云端API」双模式**：默认关闭云端，完全断网纯本地运行；仅打开时为视频镜头分析调用云端，最小化数据上云，保护隐私。
2.  **对话框式电脑全控制**：文本/语音指令触发文件管理、软件启停、键鼠模拟、系统指令（关机/录屏/进程管理），无需手动操作。
3.  **轻量化视频分析**：本地读视频+抽关键帧，云端分析输出**结构化文字版镜头脚本**，不做任何视频编辑/生成，内存占用≤5%。
4.  **场景化多窗口交互**：多窗口绑定独立场景（编程/创作/办公），单窗口+全局双层记忆，加密存储，支持自然语言检索。
5.  **极致轻量化**：安装包≤200MB，内置所有依赖，一键安装/卸载无残留，单窗口CPU/内存≤15%，适配Windows10/11+Ubuntu，预留macOS接口。

## 📌 核心功能模块
### 已规划（按开发优先级推进）
#### 第一优先级（核心基础，即将开发）
- [ ] 对话框核心交互：多窗口+自定义（外观/快捷键/标签）+悬浮球+本地控制台
- [ ] 电脑本地控制：文件/软件/键鼠/系统指令全维度控制
- [ ] 多模态基础：本地图片OCR/语音ASR-TTS/视频本地读帧+关键帧抽取
- [ ] 云端API开关：手动控制，默认关闭，仅视频分析调用云端
#### 第二优先级（功能补全）
- [ ] 本地记忆管理：单窗口/全局/场景分类记忆，加密存储+自然语言检索
- [ ] 基础任务调度：自然语言任务拆解+定时/周期任务
- [ ] 视频云端分析：对接豆包云端API，输出结构化文字镜头脚本
### 暂不集成（后续迭代）
- ❌ 视频/图片AI生成/编辑/特效（重计算，偏离轻量化定位）
- ❌ 多端/云端同步（牺牲隐私）
- ❌ 复杂插件系统（增加资源占用）

## 🔍 与同类开源项目的核心差异
| 对比维度 | 白城主 | OpenClaw（外接API） | 其他本地AI助手（LocalAI） |
|----------|--------|---------------------|---------------------------|
| **Token成本** | 本地核心功能0成本，云端补充单次0.5-0.6元 | 个人高频30-100元/月，重度200-800元/月 | 本地运行0成本，功能单一 |
| **电脑控制能力** | ✅ 对话框指令全维度控制，直接执行操作 | ✅ 可生成控制代码，需手动执行 | ❌ 仅AI交互，无控制能力 |
| **云端API控制** | ✅ 手动开关，仅视频分析调用 | ❌ 全功能依赖云端，无可控开关 | ❌ 纯离线/默认云端，无可控开关 |

| **离线可用** | ✅ 本地核心功能离线可用 | ❌ 完全依赖网络 | ✅ 纯离线运行 |
| **隐私性** | ✅ 本地数据全程不上传，仅视频关键帧按需上云 | ❌ 所有操作数据上传云端API | ✅ 纯本地运行，无数据泄露 |
