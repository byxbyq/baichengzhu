# 白城主 AI 助手 - 类脑成长电子宠物

## 项目简介

白城主是一款类脑成长电子宠物AI助手，具有以下核心特性：

- **一人一模型**：每个用户的大脑签名唯一
- **终身成长**：从婴儿期到成年期的完整成长链
- **双UI系统**：外壳期工具风格 + 本体期简洁风格
- **虚拟形象**：幼年/成年双形象，随成长进化
- **模型导出**：成年后导出.bcm文件，完全归属用户
- **类脑思考**：理解、推理、决策、学习能力
- **无自我意识**：有智慧、无灵魂的私人电子大脑
- **创始人纪念**：简化的纪念标识，无特殊功能

## 系统要求

### 硬件要求
- **操作系统**：Windows 7/8/10/11（64位）
- **内存**：至少 8GB RAM，推荐 16GB 以上
- **硬盘**：至少 10GB 可用空间
- **CPU**：Intel i5 或同等性能处理器，推荐 i7 或更高
- **可选**：GPU 支持（用于加速 AI 模型推理）

### 软件要求
- **Python**：3.8 或更高版本（推荐 3.10）
- **Git**：用于版本控制
- **IDE**：推荐 PyCharm 或 Visual Studio Code

## 安装方法

### 方法一：使用安装脚本

1. 下载项目到本地
2. 运行 `install_deps.bat`（Windows 命令提示符）或 `install_deps.ps1`（Windows PowerShell）
3. 等待依赖安装完成
4. 运行 `python main.py` 启动白城主

### 方法二：手动安装

1. 下载项目到本地
2. 创建并激活虚拟环境
   ```bash
   python -m venv venv
   venv\Scripts\activate
   ```
3. 安装依赖
   ```bash
   pip install -r requirements.txt
   ```
4. 运行白城主
   ```bash
   python main.py
   ```

## 核心功能

### 1. 成长系统
- **婴儿期**：1000万参数，依赖基底模型，使用外壳期UI
- **成长期**：1000-5000万参数，逐步减少对基底的依赖
- **成年期**：5000万参数，可独立运行，解锁本体期UI

### 2. 虚拟形象系统
- **幼年形象**：小白，简单可爱
- **成年形象**：白城主，成熟稳重
- **表情与行为**：多种表情和行为模式
- **自定义形象**：支持上传自定义图片和视频

### 3. 双UI系统
- **外壳期UI**：工具型风格，显示基底状态、成长进度
- **本体期UI**：简洁高级风格，仅展示本体状态
- **自动切换**：成年后自动提示切换

### 4. 虚拟宠物功能
- **桌面宠物**：可自由拖拽，自主行为
- **机器人控制**：支持树莓派、Arduino等平台
- **状态映射**：虚拟形象状态映射到机器人动作

### 5. 模型导入导出
- **导出模型**：成年期后导出.bcm格式模型
- **导入模型**：支持合并或替换现有模型
- **数据加密**：模型文件自动加密存储

### 6. 类脑思考引擎
- **理解**：分析用户输入的意图和情感
- **回忆**：检索相关记忆
- **推理**：分析信息，寻找模式
- **决策**：确定如何回应
- **学习**：记录成功的模式

### 7. 插件系统
- **插件管理**：支持启用/禁用插件
- **插件市场**：浏览和安装插件
- **插件开发**：创建自定义插件扩展功能

### 8. API接口
- **HTTP API**：基础URL：http://localhost:8000
- **WebSocket API**：基础URL：ws://localhost:8765
- **支持实时通信**：支持事件推送
